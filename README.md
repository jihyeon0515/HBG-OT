# 헬스보이짐 OT 관리 시스템 (hbgym_ot)

헬스보이짐 분당정자점의 신규회원 OT(오리엔테이션) 문진·배정·진행을 관리하는 **Flutter 웹앱**입니다.

## 워크플로

```
회원 문진표 작성 → 관리자 접수 → 트레이너 배정 → OT 진행/작성 → 완료 → 성공/실패
```

상태: `접수 → 배정완료 → 진행중 → 완료 → (성공 | 실패)`

## 역할별 화면

- **회원**: 신규 상담 문진표 작성(기본정보·방문 계기·운동목적·병력·운동경험·성격·특이사항) 후 제출
- **관리자**: 상태 배너로 분류 조회, 기간·종목·담당자 필터, 트레이너 배정, 트레이너별 보드, 완료건 성공/실패 처리
- **트레이너**: 배정 알림, 회원 작성 내역 조회, InBody·근력·자세·컨설팅·오티 3회차 작성 → 오티문진표 미리보기(로고·인체 이미지) → 완료

## 기술

- Flutter 3.44 (Web)
- 상태관리: provider
- 로컬 저장: shared_preferences (다기기 연동 전 단계 — 이후 Firebase 연결 예정)

## 실행

```bash
flutter pub get
flutter run -d chrome
# 또는 웹 빌드
flutter build web
```

빌드 산출물(`build/web`)을 정적 서버로 서빙하면 브라우저에서 사용할 수 있습니다.

## 데모 파라미터

- `?seed=1` : 샘플 데이터 시드
- `?role=member|admin|trainer` : 시작 역할
- `?view=list` : 관리자 화면을 목록형으로 시작
